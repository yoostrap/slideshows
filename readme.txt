== Installation ==
1. Navigate to your WordPress Dashboard.
2. Go to Plugins -> Add New.
3. Search for **Hizzle Slideshows**.
4. Click "Install Now" and then activate the plugin via the "Activate Plugin" link.

== Features & Settings ==
- Add unlimited slides effortlessly.
- Seamlessly integrate any block into your slides.
- Live preview your slides directly in the editor.
- Ensure responsiveness across all devices.
- Customize slides per view for optimal display.
- Navigate with ease using prev/next buttons.
- Utilize Autoplay for dynamic presentations.
- Customize appearance to match your brand.
- Toggle pagination visibility as per your preference.
